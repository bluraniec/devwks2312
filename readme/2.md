### 1st step - Create Database
In the CLI issue command influx to run the InfluxDB:
```
root@ubuntu:~/# influx
```
Create database TELEMETRY_DATA where you will store all collected data: 
```
> create database TELEMETRY_DATA
```
You can close it now:
```
> exit
```
